# MD5 加密

> MD5是一个安全的散列算法，
>
> 输入两个不同的明文很难得到相同的输出值，
>
> 根据输出值，不能得到原始的明文，
>
> 即其过程不可逆；

> 破解：字典、穷举
>
> 加盐，增加复杂度

> 默认32位
>
> 需要16位时，截取8到24（不含）字符串

# SHA 加密 （安全哈希算法）

> 单向散列算法
>
> 接收一段明文，以不可逆的方式将它转换成一段位数固定的密文
>
> SHA-1/SHA-224/SHA-256 适用 不超过 2^64 二进制位
>
> sha-384/sha-512 适用 不超过 2^128 二进制位
